## Running Locally

Clone the project

```bash
git clone https://github.com/Santiago-Tesla/Terminal-Portfolio.git
```

Go to the project directory

```bash
cd Terminal-Portfolio
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
